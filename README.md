# ES6 `Number.isInteger()` polyfill

## Installation

```sh
npm install number.isinteger
```

Or

```sh
yarn add number.isinteger
```

## Usage

```js
require('number.isinteger');

Number.isInteger(42); //? true
Number.isInteger(3.1415); //? false
```
